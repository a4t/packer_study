# Packerのインストール

## インストール
### Mac

```
$ wget https://releases.hashicorp.com/packer/1.2.2/packer_1.2.2_darwin_amd64.zip
```

### Linux

```
$ wget https://releases.hashicorp.com/packer/1.2.2/packer_1.2.2_darwin_amd64.zip
```

### 共通

```
$ unzip packer_1.2.2_darwin_amd64.zip
$ rm -rf packer_1.2.2_darwin_amd64.zip
$ mv packer [path/to/packer]
```

## 動作確認

```
$ packer version
Packer v1.2.2
```
